## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/GuilhaoF/biscoito-app-.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install ou yarn install
```

Inicie o servidor

```bash
  expo start 
```
